List of Features

(I can't wait to see how short it is)

General Features

  • Very nice looking error messages
  • Floating point math
  • Integer math
  • Pointer dereferencing
  • Full set of operators, including bitwise and logical operators
  • Type system
  • Variables (Local or global)
  • for loops, and if statements (More coming soon)
  • Can call C functions out of any DLL
  • Can be called by C/AHK when compiling to a DLL
  • Fully conforms to the x64 calling convention
  • Speed (not as fast as C, but much faster than AHK)
  • Obfuscation (by nature, since there's no decompiler for this, but the generated code is somewhat readable)
  • Builds lightweight files (Since there's no need for a very large runtime, only like ~300 bytes of boilerplate is added to a program)

Syntax Features

  • A verbose (but not overly wordy) syntax for defining and importing functions
  • Short (and easy to type) type names
  • No statement terminator character (Most C-like languages have ;, Relax has no character like that)
  • Most statements can be single line (through just moving the code all onto the same line)
  • Separate assignment and comparison operators (:= to assign, = to compare)
  • // and /* whatever */ comments
  • Binary, octal, hex, and decimal literals (through the 0b, 0o, and 0x prefixes)
  • All whitespace but \n is ignored, so you can format your code however you'd like (Can you tell I'm running out of ideas for this list?)
  • String and character literals (through "string" and 'c')

Compiler Features

  • Attempts to optimize code
  • Generates pure x86-64 machine code
  • Uses registers for nearly all operations
  • Generates surprisingly readable machine code
  • Uses dedicated 0/1 registers for improved code density (compared to using 0/1 as immediate values)
  • Has floating point support
  • Doesn't need to recompile standard library unless there is an update
  • Selects smallest instructions for most operations
